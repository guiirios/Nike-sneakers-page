


# 🏀 Nike Store Clone

Projeto de front-end inspirado na Nike, desenvolvido com HTML e CSS puro, focado em layout moderno, responsividade e prática de estruturação de páginas web.

---

## 📸 Preview

![Preview do projeto](assets/img/preview.png)

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Font Awesome (ícones)
- Design responsivo (media queries)

---

## 💡 Funcionalidades

- Header com menu de navegação
- Barra de busca estilizada
- Banner principal com imagem e texto sobreposto
- Grid de produtos (cards de tênis)
- Layout responsivo para mobile
- Footer completo com links e redes sociais
- Ícones de pagamento
